zenitai.transform.WoeTransformer¶

class zenitai.transform.WoeTransformer(min_sample_rate: float = 0.05, min_count: int = 3, save_data: bool = False, join_bad_categories: bool = False)¶

Bases: sklearn.base.TransformerMixin, sklearn.base.BaseEstimator

Класс для построения и применения WOE группировки к датасету

Parameters

  • min_sample_rate (float, default 0.05) – Минимальный размер группы (доля от размера выборки)

  • min_count (int, default 3) – Минимальное количество наблюдений каждого класса в группе

  • save_data (bool, default False) – Параметр, определяющий, нужно ли сохранить данные для обучения трансформера внутри экземпляра класса

  • join_bad_categories (bool, default False) –

    Определяет, должени ли трансформер предпринять попытку для объединения катогориальных групп в более крупные

    Warning

    join_bad_categories - Экспериментальная функция. Способ группировки категорий нестабилен

plot_woe(predictors=None)¶

Отрисовка одного или нескольких графиков группировки

Parameters
