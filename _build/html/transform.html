Агрегация данных по значениям предиктора. Рассчитывает количество наблюдений, количество целевых событий, долю группы от общего числа наблюдений и долю целевых в группе

Таблица данных для агрегации, должна содержать поля „predictor“ и „target“. Поле target при этом должно состоять из 0 и 1, где 1 - целевое событие

low_accint, default None

Параметр для округления значений предиктора. Если None, то предиктор не округляется. Если целое неотрицательное число, параметр используется для определения количества знаков после запятой, остальные значения игнорируются

ypandas.Series

Mассив меток класса (0, 1)

Категориальные значения (пустышки и несравнимые значения). Элементы списка должны быть строками
